# Advent of Code

Solving http://adventofcode.com puzzles using JavaScript.

## To run a solution

`npm start <year#> <day#> [<part#>]`

## To test solutions

`npm test [<year#>]`

## To set up a new solution

`npm run setup <year#> <day#>`
